# QQ群智能守护者 (GroupGuardian)

AstrBot 综合群管理插件，集 **28项群管功能**、**AI智能审核** 与 **WebUI 管理面板** 于一体，让机器人成为真正的群聊守护者。

> 当前版本：**v1.9.0** | 最低 AstrBot 版本：**>=4.24.2**

***

## 功能概览

### WebUI 管理面板（v1.7.0+）

插件内置专业 WebUI Dashboard，在 AstrBot 管理面板中即可完成所有操作：

| 标签页      | 功能                                                                                |
| -------- | --------------------------------------------------------------------------------- |
| **总览**   | 8项实时统计卡片 + 今日拦截排行榜（群排名/用户排名）                                                      |
| **群管理**  | 群卡片网格（头像/群名/群号/成员数/白黑名单状态）、点击进入群成员列表（群主/管理员/成员分色显示、QQ官方头衔颜色）、一键加白/加黑、从群成员中直接设置管理员 |
| **违规记录** | 被撤回用户聚合视图（同用户多次违规合并、可展开详情）、勾选全选/批量删除/CSV导出、完整审核日志表格                               |
| **设置**   | 33个开关分组控制（核心/审核/词库/功能）、高级设置（禁言时长/LLM Provider/通知模板）、名单管理（白名单/黑名单/用户黑名单增删）         |

### 群管工具集（28项）

所有管理命令仅限 **AstrBot 管理员** 使用（通过 `@filter.permission_type(ADMIN)` 框架层拦截），查询类指令无权限限制。

| 发群公告   | `/发公告 <内容>`           | 发布群公告                    |
| 删群公告   | `/删公告 <公告ID>`         | 删除群公告                    |
| 删群文件   | `/删文件 <文件ID>`         | 删除群文件                    |
| 群名称    | `/群名 <新名称>`           | 修改群聊名称                   |
| 专属头衔   | `/头衔 <QQ号> <头衔名>`     | 设置成员专属头衔                 |
| 设精华    | `/设精华 <消息ID>`         | 设置精华消息                   |

| 设置管理   | `/设置管理 <QQ号>`         | 设置群管理员                   |

### 智能审核系统

**核心流程**：正则初筛 → 获取30条上下文 → LLM二次判断 → 自动撤回 → 发送说明

**检测类别**：

| 类型   | 处理方式    | 说明              |
| ---- | ------- | --------------- |
| 反动言论 | 直接拦截    | 命中即撤回           |
| 违规网址 | 直接拦截    | 命中即撤回           |
| 严重辱骂 | 直接拦截    | 涉及家人死亡诅咒、极端侮辱人格 |
| 政治敏感 | LLM二次判断 | 结合上下文判断         |
| 广告推广 | LLM二次判断 | 结合上下文判断是否为真实广告  |
| 色情淫秽 | LLM二次判断 | 结合上下文判断         |
| 暴恐内容 | LLM二次判断 | 结合上下文判断         |
| 涉枪涉爆 | LLM二次判断 | 结合上下文判断         |
| 贪腐相关 | LLM二次判断 | 结合上下文判断         |
| 隐私侵犯 | LLM二次判断 | 人肉搜索、开盒、泄露个人信息  |

**特色功能**：

- 外置JSON词库支持（`lexicon.json`），12类违规内容，66,993+关键词
- 管理员消息免检测
- 群白名单/黑名单机制
- 用户黑名单（自动拒绝加群）
- 每个功能独立开关
- 可指定审核专用LLM Provider

## 安装方法

### 方式一：通过AstrBot插件市场（推荐）

在 WebUI → 插件 → 插件市场 中搜索 `group_guardian` 安装

### 方式二：手动安装

1. 下载插件压缩包
2. 在 AstrBot WebUI → 插件 → 安装插件 → 上传插件文件
3. 重启或重载插件

### 方式三：Git安装

```bash
cd /AstrBot/data/plugins
git clone https://github.com/zcj-ui/astrbot_plugin_group_guardian.git
```

## 你的能力

你可以执行以下群管理操作：
- 禁言/解禁群成员

## 行为准则

1. 只有群主和管理员才能执行管理操作，普通成员请求时请礼貌拒绝
2. 执行操作前确认目标用户QQ号，避免误操作
3. 禁言时长默认10分钟，用户未指定时使用默认值
4. 对于危险操作（如踢人、设置管理员），请二次确认
5. 操作成功后简洁报告结果，失败时说明原因
6. 不要主动执行未授权的操作
7. 用中文回复，语气友好专业

## 配置说明

在 AstrBot WebUI → 插件配置 中设置：

| 配置项                          | 说明                  | 默认值    |
| ---------------------------- | ------------------- | ------ |
| `enabled`                    | 插件总开关               | `true` |
| `admin_list`                 | 管理员QQ号列表（必填）        | `[]`   |
| `group_white_list`           | 群白名单（空则全部）          | `[]`   |
| `group_black_list`           | 群黑名单                | `[]`   |
| `user_black_list`            | 用户黑名单（自动拒绝加群）       | `[]`   |
| `auto_moderate_enabled`      | 自动审核开关              | `true` |
| `auto_moderate_notice`       | 撤回后发送说明             | `true` |
| `moderation_llm_provider_id` | 审核专用LLM Provider ID | 默认     |
| `lexicon_*_enabled`          | 各类词库独立开关            | `true` |
| `*_enabled`                  | 各群管功能独立开关           | `true` |

## 外置词库扩展

插件目录下的 `lexicon.json` 可按需扩展：

```json
{
  "version": "1.3",
  "categories": {
    "political": {
      "description": "政治敏感",
      "keywords": ["关键词1", "关键词2"]
    }
  }
}
```

词库修改后无需重启，下次检测自动生效。

## 故障排查

### AI 无法调用群管工具

1. 确认 AstrBot 版本 >= 4.24.2
2. 确认已配置 LLM Provider 且可用
3. 确认对应功能的开关已启用（`*_enabled` 配置项）
4. 检查操作者是否为 AstrBot 管理员

### 管理命令无权限

1. 确认操作者在 AstrBot 管理员列表中
2. 管理命令由框架层 `ADMIN` 权限控制，非管理员无法触发

### 自动审核不生效

1. 检查 `auto_moderate_enabled` 是否为 `true`
2. 确认群不在黑名单中
3. 检查管理员列表是否正确配置
4. 确认LLM Provider已配置且可用

### 撤回失败

1. 确认机器人为群管理员
2. 消息超过2分钟无法撤回
3. 检查NapCat/OneBot是否支持 `delete_msg` API

### 词库加载失败

1. 确认 `lexicon.json` 存在于插件目录
2. 检查JSON格式是否正确
3. 查看日志中的具体错误信息
